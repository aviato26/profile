The Coffee Stop

After seeing this awesome scroll animation from codegrid, i had to try to recreate this frontend theme. So after cropping the cup, stacking it on top of the original image and updating the original images transform property with the scale method being updated by a little bit of vanilla JS, and voila!

Talk Box

This project was built using the react, sqlite, sequelize and css grid. It will show any device within about a hundred feet, once both people accept to chat they are connected to a private chat room

Movie Bin

This app was built using the MERN stack. app will search movies and if one one that the user would like to see later it can be added to there list to be viewed later
